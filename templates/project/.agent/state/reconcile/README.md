# Reconcile Artifacts

This directory stores the latest cross-role comparison outputs.

Expected artifacts:

- `latest.json`
- `latest.md`

These summarize:

- agreements
- conflicts
- missing validations
- unresolved risks
- missing or partial role outputs
- final readiness state

These files are a portable repo-local backstop for any tool that needs to continue work without relying on machine-local memory.
