# Kiwi Control Public Demo Video Package

This folder is the recording package for a stronger public Kiwi Control demo video.

The goal is a 2-4 minute video that shows:

1. a real repo
2. real Kiwi setup
3. real Kiwi UI and CLI surfaces
4. a real task
5. where Claude Code fits
6. the measured A/B proof from the completed Markdown Notes Organizer experiment

## Record From

Use Repo B as the main demo repo:

```bash
cd "/Volumes/shrey ssd/my ssd-playground/test-B"
```

Use Repo A only for comparison evidence:

```bash
cd "/Volumes/shrey ssd/my ssd-playground/test-A"
```

## Primary Assets

- `shot-list.md` - exact visual sequence
- `script.md` - narration outline
- `commands.md` - terminal commands to run during recording
- `captions.md` - on-screen captions/lower thirds
- `proof-summary.md` - measured A/B claims and source links
- `chapter-markers.txt` - timeline markers
- `title-card.txt` - opening card copy

## Honesty Constraints

- The A/B result is one controlled run, not a universal benchmark.
- The proof uses direct Claude JSON usage data.
- Do not claim certain or universal savings.
- Do not show `.env`, private dashboards, or secret-bearing windows.
- If a command fails during recording, stop and fix before continuing.

## Final Video Shape

One strong main video is better than many weak clips. Aim for 2-4 minutes.
