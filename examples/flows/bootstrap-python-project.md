# Bootstrap Python Project

```bash
node dist/cli.js bootstrap --target "/path/to/python-project" --project-type python --dry-run
node dist/cli.js bootstrap --target "/path/to/python-project" --project-type python
node dist/cli.js run "stabilize input validation for the upload path" --target "/path/to/python-project" --profile product-build --mode assisted
```

Use this when you want bootstrap to bias the starter profile, specialists, and validation hints toward a Python-style repo from the first run.
