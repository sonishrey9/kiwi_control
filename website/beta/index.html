Beta limitations

What is still rough

Kiwi Control is a public beta. Core local-first repo control is real, but desktop trust, some UX flows, and platform distribution polish are still in progress.

Desktop trust

Windows signing and macOS notarization may still be incomplete for some beta releases. Check release notes and checksums before trusting an installer.

Workflow blocks

Blocked validation, checkpoint, and handoff states are often real repo-state problems, not UI failures. Kiwi should explain recovery commands, but it will not silently bypass repo-local truth.

Tool/runtime parity

Do not assume universal parity across every editor, assistant, or hosted runtime. Kiwi is repo-first and local-first by design.

Contributor caveat

Some external macOS volumes create ._* files inside .git. That can corrupt large clones and break Git pack indexes.
