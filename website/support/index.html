Support

Get help without losing the release trail.

Kiwi Control support is still best-effort during beta, but the install and release surface is stable: this website tracks the current public release metadata, GitHub keeps release notes and release history, and support starts with enough environment detail to reproduce the issue.

Before opening an issue

Include your OS and shell.
Include your Node version if you installed the CLI manually.
Say whether you ran from source or a release artifact.
Include the exact command you ran.
Say whether the repo was already initialized.

Support channels

Bugs and reproducible regressions: GitHub Issues.
Product discussion: GitHub Discussions when enabled.
Security-sensitive reports: use private reporting, not public issues.
Creator contact: Shrey Soni.

External volume caveat

If you are using an external macOS volume, mention that first. AppleDouble ._* files can corrupt Git metadata and make debugging look stranger than it is.
