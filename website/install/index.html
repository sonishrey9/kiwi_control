Install Kiwi Control

GitHub Releases is the source of truth for public beta binaries. Windows and macOS desktop installs are the normal-user product path.

CLI

1. Download the CLI bundle from GitHub Releases.
2. Run install.sh or install.ps1.
3. Use kiwi-control --help or kc --help.

Desktop

1. Download the matching desktop bundle from the same release.
2. Install it normally for your platform.
3. Launch Kiwi once and use the onboarding flow to install kc and choose a repo.

First repo flow

kc init if the repo is not initialized
kiwi-control status
kiwi-control prepare "your real task"
kiwi-control validate

Beta warning

If signing/notarization is incomplete for your platform, expect system trust warnings. Always check release notes and checksums.
