Install guide
Latest release: v0.2.0-beta.1

Install Kiwi Control with the desktop app first, optional terminal commands second.

Cloudflare hosts the public installers, checksums, and release manifest. GitHub keeps the release notes and release history. Use this page when you want the full desktop-first flow and the optional `kc` verification steps.

Back to downloads Open all release assets

Desktop install

1. Download the macOS or Windows desktop installer from the homepage or all downloads.
2. Install Kiwi Control normally for your platform.
3. Launch the app, choose a repo, and initialize it if needed.
4. Keep using the desktop app. Terminal commands remain optional.

Optional CLI enablement

1. Open the app after install.
2. Choose Enable terminal commands (kc) only if you want the Terminal path too.
3. Open a fresh terminal.
4. Verify with the commands below.
