Desktop install

1. Download the macOS or Windows desktop installer from the homepage or all downloads.
2. Install Kiwi Control normally for your platform.
3. Launch the app once so Kiwi can finish default terminal command setup and verification.
4. Choose a repo, initialize it if needed, and keep using desktop or kc.

Default terminal command setup

1. Installed desktop builds auto-attempt kc setup by default.
2. macOS may ask for administrator approval on first launch to write command shims into /usr/local/bin.
3. Windows is configured for machine-wide installer behavior, and Kiwi verifies kc from a fresh PowerShell process when run on a real Windows host.
4. Open a fresh terminal and verify with the commands below.
