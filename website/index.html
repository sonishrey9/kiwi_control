Repo-local control for coding agents

Kiwi Control is a thin repo-local control plane for coding agents. It keeps authority in the repository, exposes an operator-friendly CLI, and ships a Tauri desktop shell for visibility, validation, and workflow control.

What Kiwi is

Repo-local authority, explicit artifacts, deterministic CLI workflows, and a desktop shell that reflects repo state instead of inventing hidden state.

What Kiwi is not

Not a hidden cloud runtime, not a generic autonomous agent platform, and not a product that silently overrides repo-local truth.

Install path

Use the CLI and desktop bundles from GitHub Releases. Use kiwi-control and kc as the primary commands.

Beta trust

Desktop signing and notarization status must be checked in each release. Do not assume every artifact is fully trusted yet.
