Install in four steps

1. Download the desktop installer for macOS or Windows.
2. Install Kiwi Control normally for your platform.
3. Launch the app and choose a repo.
4. Optionally enable terminal commands later if you want `kc` from Terminal.

Optional CLI later

You do not need terminal setup for normal desktop use. If you want the power-user path too, use Enable terminal commands (kc) after install or grab the manual CLI bundles from all downloads.

Trust still matters

Cloudflare hosting makes downloads public, not trusted. Always check the current release notes, checksums, and platform trust status before you trust an installer.

After install

Verify optional terminal commands

Desktop use works immediately. If you later enable `kc`, open a fresh terminal and verify it.

macOS: command -v kc
macOS: kc --help
Windows: Get-Command kc
Windows: kc --help
