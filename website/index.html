Install in three steps

1. Download the desktop installer for macOS or Windows.
2. Install Kiwi Control normally for your platform.
3. Launch the app and let onboarding install kc for the repo flow.

Prefer CLI?

The CLI bundle is still available from GitHub Releases. Manual CLI install instructions live on the install page, below the desktop-first flow.

Beta trust posture

Always check release notes and checksums before trusting an installer. Signing and notarization status may vary by release.

Detailed install

CLI and manual setup

Desktop is the primary path, but the CLI bundle still works when you want a direct shell-based install.

1. Open all downloads or the latest release page.
2. Download the CLI bundle for your OS.
3. Run install.sh or install.ps1.
4. Use kiwi-control --help or kc --help.

Open full install guide

Before you install

Know the current beta limits

Kiwi Control is already useful for local-first repo control, but platform trust, signing, and some polish items are still in progress.

Desktop signing and notarization are still release-specific checks.
GitHub Releases is the authoritative artifact source.
Repo-local state remains the product truth, even when the app looks polished.
