# Kiwi Control A/B Proof Bundle

This directory contains the measured artifacts used by the public proof/demo surface.

Scope:
- one controlled greenfield A/B run of the same Markdown Notes Organizer task
- same machine
- Repo A without Kiwi workflow help before implementation
- Repo B with Kiwi workflow help before implementation
- direct Claude JSON usage data captured from the CLI

Use these files as raw evidence, not as a universal benchmark.
